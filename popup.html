ZapSearch - Spotlight Search for Chrome

Current Shortcut to launch: Not Found. Click 'Change Shortcut' Button ASAP

Note: After installation reload old tabs to make ZapSearch working

Default Shortcut to launch

"Ctrl+Shift+L" or "Command+Shift+L"

Usage - Prepend following characters

> To run Commands

@ To Search in History

? To Google Search

Example: '@ Stack overflow' - to search History with 'Stack overflow'

Example: 'Stack overflow' - to search Bookmarks with 'Stack overflow'
